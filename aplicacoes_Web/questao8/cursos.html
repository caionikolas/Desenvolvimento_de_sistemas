HTML

O que é?

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Sobre o curso:

Este curso ensina desde dos conceitos básicos da linguagem HTML, ate a criação de formularios para a Web, ideal para iniciantes, o par quem quer um guia de consultas, também possui um Glossário com todas as palavras mais utilizadas na internet

DreamWeaver

O que é?

é um software de desenvolvimento voltado para a web criada pela Macromedia, e que está atualmente na versão CC.

Sobre o curso:

O treinamento dado no curso é voltado para criação completa de sites usando banco de dados mysql, css3, html5 e php, no curso montamos um site completo passo a passo, com o curso o aluno terá todo conhecimento para criar sites e começar a ganhar sua própria renda.

Flash

O que é?

é um software primariamente de gráfico vetorial - apesar de suportar imagens bitmap e vídeos - utilizado geralmente para a criação de animações interativas que funcionam embutidas num navegador web e também por meio de desktops, celulares, smartphones, tablets e televisores.

Sobre o curso:

você vai aprender a criar Banners Animados em Flash , Banners muito profissionais e ficar totalmente preparado já para entrar em qualquer empresa de designer .

Fireworks

O que é?

é um editor de imagens de bitmap e desenho vetorial desenvolvido pela Macromedia, posteriormente adquirido pela Adobe.

Sobre o curso:

Neste curso os alunos vão aprender tudo para começar a trabalhar com o desenvolvimento desta ferramenta de design. Os usuários poderão desenvolver seus próprios Layouts. Além disso, o curso também ensina os estudantes a fazer a integração com flash e dreamweaver.
